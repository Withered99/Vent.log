# Vent.Log

## Info
Vent.Log is a catalogue of Modded Among Us roles from various mods. It is currently still being set up but will be finished soon.

As of right now all information (other than vanilla) comes from [Town of Us: Mira](https://github.com/AU-Avengers/TOU-Mira) but will be updated in the future.
